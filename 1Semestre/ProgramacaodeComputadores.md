# Programação de computadores (9/10)
**Resumo da matéria:**
* Esta matéria buscou ensinar lógica de programação e dependendo do professor, a linguagem de programação e a forma de ensinar principalemnte irá mudar. Comigo a professora passou linguagem C!

**Anotações da matéria e sobre o que é:**
* Lógica de programação.
* Linguagem C, mas para outros alunos de outras turmas teve opção de java e python também.
* Buscou ensinar o básico como:
  * Tipos de variáveis.
  * Declaração de variáveis.
  * Saida e entrada de dados.
  * Estruturas condicionais.
  * Estruturas de repetição.
  * Variaveis compostas.
***
Este foi um resumo sobre o que trata-se na matéria programação de computadores!
