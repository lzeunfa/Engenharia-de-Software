**Anotações da matéria e do que se trata:**
* Algoritmo:
  * Sequência de instruções.
* Programa:
  * Conjunto de instruções codificadas em linguagem ded programação.
* Sistema computacional:
  * Conujunto de componentes físicos(hardware) e lógicos(software) que trabalham juntos.
  * Para Tanenbaun:
    * Hardware: Componente físico do sistema.
    * Sistema operacional: Gerenciamento do hardwware e fornece interface.
    * Aplicativos: Software que é executado no sistema para realizar tarefas.
  * Fluxo de processo de montagem:
    * Programa fonte-> montador -> linguagem de máquina.
    * Montador: Tradutor, traduz o código para linguagem de baixo nível e distantes da linguagem humana (assembly).
    * Linguagem de máquina: binários.
    * Linguagem de alto nível é mais próxima do usuário. <br>

| Ferramenta     | Nível       | Características                                              |
|----------------|-------------|--------------------------------------------------------------|
| Montador       | Baixo nível | Trabalha próximo ao hardware; gera código máquina diretamente. |
| Compilador     | Médio/Alto  | Tradução completa antes da execução; geralmente mais rápido; gera um único arquivo executável. |
| Interpretador  | Médio/Alto  | Tradução linha a linha em tempo de execução; mais flexível e mais fácil de depurar. |
