# Programação Web (10/10)
**Resumo da matéria:**
* A matéria de programação web ensinou a base para criação de projetos para web utilizando html, css, bootstrap, js, php e MySQL.
   A faculdade claro não tem como lhe ensinar tudo, mas o que foi passado como interação com banco de dados com queries criadas via php,
   interação com formulários dentro do site e estruturação do site foram bem importantes e agregaram demais aos conhecimentos que eu já tinha
   e os que eu não tinha!
***
**O que foi passado nesta matéria:**
* Tecnologias Utilizadas:
  * HTML
  * CSS
  * Bootstrap
  * JS
  * PHP
  * XAMPP
  * MySQL

* As aulas desta matéria passaram pelo html, css e bootstrap de forma bem rápida com intuito de ensinar o básico sobre as estruturas e classes. O foco maior seria o php e banco de dados MySQL então o professor sugeriu que caso tivesse interesse ou duvida buscar alguns vídeos no youtube sobre html e css básico.
* Após ensinar:
  * HTMl:
    * Corpo do documento html.
    * Tags h1 até h6.
    * Tags de parágrafos p.
    * Tags de links.
    * Tags semânticas como footer, header, main.
    * Como inserir arquivos externos.
  * CSS com bootstrap:
    * Como usar o bootstrap.
    * Estilização básica com css.
    * Estrutura pré prontas do bootstrap.

* Iniciou conteúdo de PHP e banco de dados:
  * PHP:
    * Declaração de variáveis.
    * Tipos de variáveis.
    * Funções nativas.
    * Estruturas condicionais.
    * Estruturas de repetição.
    * Métodos get e post.
    * Interação com formulários html.
  * MySQL:
    * Tipos de dados.
    * Queries SQL.
    * Criação de bancos de dados.
    * Criação de tabelas.
    * Inserção, remoção e edição de dados.
    * Queries de consultas.
    * CRUD (Create, read, update and delete) juntando php e mysql.
* No fim do semestre foi passado o projeto de avaliação final em que os alunos deveriam criar um crud com interação de no mínimo duas tabelas, onde criei um sistema de sáude básico! (código disponível no meu github).
 
***
Este foi um resumo sobre o que trata-se na matéria programação web! <br>
